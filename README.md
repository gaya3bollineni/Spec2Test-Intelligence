# Spec2Test Intelligence

Spec2Test Intelligence is an AI-powered requirement-to-test case generation system that transforms acceptance criteria into structured, traceable, and explainable QA test cases.

## Current Goal
Build an MVP that:
- accepts acceptance criteria as input
- parses the requirement text
- generates structured test cases
- exports results in JSON and Excel
- provides a simple Streamlit UI

## Planned Modules
- Input normalization
- Criteria parsing
- Scenario expansion
- Expected result generation
- Export engine
- UI layer
